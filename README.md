# PvZ-Tree-Auto-Feed
PvZ Replanted - Automatically buys tree food and feeds the wisdom tree
---
Requires AutoHotkey
* For 1080p fullscreen/windowed (not borderless) by default
* Feeds and buys in batches of 10
* Number of cycles changed by editing the .ahk file
* F1 to start and F2 to stop

You can use AHK's Window spy to recalibrate the script for PvZ GOTY
