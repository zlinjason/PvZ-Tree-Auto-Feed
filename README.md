# PvZ-Tools
PvZ Replanted - Tools to automate part of the gameplay progress
---
Requires AutoHotkey v2
* For 1080p fullscreen/windowed (not borderless) by default
* You can use AHK's Window spy to recalibrate the script for PvZ GOTY
---
WisdomTree.ahk
* Feeds and buys in batches of 10
* Number of cycles changed by editing the .ahk file
* F1 to start and F2 to stop
---
EndlessAuto.ahk
* Automatically adjusts game speed using Tab, the default keybind for in-game speed
* For a 10-cob setup
* Ice-shroom timing for pool ambush zombies in the even waves
* Blover as needed for balloon zombies
* Doom-shroom planting in the pool for emergency firepower
* Autopicking seed packets at end of rounds, which also resets the game speed
* Ctrl+LButton to determine coordinates of the game window
